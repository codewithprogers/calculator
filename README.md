# calculator
This project is a simple calculator capable of performing basic math equations
such as addition, subtraction, multiplication, and division. The calculator
includes a responsive layout and handles input validation.

## Features
- Basic math operations: +, -, *, /
- Clear and reset functionality
- Undo button to fix mistakes
- Handles decimal numbers
- Displays the current calculation and result

## Technologies
- HTML
- CSS
- Javascript

## Potential Future Improvements
- Add keyboard support for input
- Implement parentheses and order-of-operations
- Add a history panel to track previous calculations
